# Rock, Paper, Scissors Game

A simple, interactive Rock, Paper, Scissors game built to run in the terminal/browser. Play against the computer and see who wins!

## 🎮 Features
* **Interactive Gameplay:** Play multiple rounds against an automated AI.
* **Score Tracking:** Keeps track of wins, losses, and ties throughout the session.
* **Input Validation:** Prevents invalid moves and guides the user.

## 📜 Rules of the Game
* **Rock** beats Scissors.
* **Paper** beats Rock.
* **Scissors** beats Paper.
* Identical choices result in a **Tie**.
